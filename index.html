GitHub + Vercel

自动化部署流程 · 无需编写 CI/CD 配置文件
1
📝 准备代码仓库
将你的项目代码推送到 GitHub 仓库，支持任何静态网站或前端框架
git init
git add .
git commit -m "first commit"
git push -u origin main
2
🔗 连接 Vercel
访问 vercel.com，使用 GitHub 账号登录，授权访问你的仓库
👉 vercel.com
👉 Sign up with GitHub
👉 选择 "Only select repositories"
3
⚙️ 导入项目
在 Vercel 中点击 "Add New Project"，选择你的 GitHub 仓库，点击 Deploy
自动识别框架 → 自动构建 → 自动部署
无需任何配置！✨
4
🎉 自动部署生效
以后每次 git push，Vercel 都会自动重新部署，无需任何手动操作
git push → 自动构建 → 自动部署 → 实时更新 ✨
